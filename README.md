# Don Raucho Botillería

Página web moderna y elegante para la botillería Don Raucho, un homenaje a la familia y la integridad.

## Estructura

La página contiene las siguientes secciones en orden:

1. **Origen** - Historia de Don Raucho
2. **Productos Destacados** - Catálogo de 3 productos (sin imágenes, para agregar manualmente)
3. **Ofertas** - Promociones del mes (2 ofertas)
4. **Ubicación** - Mapa e información de contacto
5. **Footer** - Teléfono, redes sociales e información legal

## Colores

- **Principal**: Negro (#000000)
- **Detalles**: Tonos en madera de roble / Amber para acentos
- **Fondo claro**: Blanco (#FFFFFF)
- **Fondo oscuro**: Gris pizarra (#1E293B)
- **Texto**: Gris pizarra y blanco según el tema
- **Imagen de fondo**: Bar elegante con efecto fixed

## Archivos

- `index.html` - Estructura HTML de la página
- `style.css` - Estilos CSS personalizados
- `script.js` - Funcionalidad JavaScript
- `img/logo.jpg` - Logo de Don Raucho

## Características

- Diseño responsive (mobile-first)
- Navbar simple con navegación suave
- Cards sin imágenes (listas para personalizar)
- Teléfono y redes sociales en el footer
- Animaciones suaves al hacer scroll
- Soporte para tema oscuro (automático con Tailwind CSS)

## Personalización

Para agregar imágenes a los productos y ofertas, reemplaza los divs grises con:

```html
<img src="tu-imagen.jpg" alt="Descripción" class="w-full h-full object-cover">
```
